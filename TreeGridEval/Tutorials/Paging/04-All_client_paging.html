4. All client paging

All client paging types - Paging, ColPaging, ChildPaging, ChildParts

Source: 04-All_client_paging.xml
  • This example just demonstrates all the paging types described in previous tutorials. All the attributes set to 2 for rendering on demand and to 1 for rendering on start with progress dialog.
  • Cfg attribute

    Paging

    creates root row pages. Supports also various pagers. Possible also server paging (value 3).
  • Cfg attribute

    ChildPaging

    controls rendering children of collapsed rows. Possible also server paging (value 3).
  • Cfg attribute

    ChildParts

    controls rendering children of expanded rows.
  • Cfg attribute

    ColPaging

    creates column pages.
  • See especially the second button on toolbar (the Repaint   /   ) to switch between rendering pages on start and rendering pages on demand / scroll (the attribute values 1 / 2).
    Remember, this grid has more than 100,000 cells, so rendering all rows on start will take some time.
