E-shop example

Demonstration of eshop done by TreeGrid. Example of using some functions from Extended API.
Demonstrates rows iteration, changing value, using events, changing colors, using html cells to display generated results.
You can change any cell to contain amount of purchased items and TreeGrid calculates price and generates sales report.
This is example of CoqSoft's eshop. If you want to purchase TreeGrid, select Purchase online from left menu.
